# AI Code Reviewer

This project analyzes Python code using:
- AST-based static analysis
- Rule-based issue detection
- Transformer-based AI explanations

## Features
- Code summary (functions, lines)
- Issue detection (long functions, deep nesting, bad variable names)
- Code scoring system
- AI-generated explanations
- Refactoring hints

## How to use
1. Open `AI_Code_Reviewer.ipynb` in Colab
2. Paste your Python code
3. Run all cells to see analysis, suggestions, and AI review
